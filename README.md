# Stonehard AI

**EN** — This project assists players in playing [Stoneshard](https://store.steampowered.com/app/625960/Stoneshard/) by using computer vision and machine learning to analyze the game state, detect obstacles, and suggest movements. It also serves as a practical application of AI knowledge.

**FR** — Ce projet assiste les joueurs dans leur partie de [Stoneshard](https://store.steampowered.com/app/625960/Stoneshard/) en utilisant la vision par ordinateur et le machine learning pour analyser l'état du jeu, détecter les obstacles et suggérer des déplacements. Il sert également d'application pratique des connaissances en IA.

---

## Stack

- Python
- TensorFlow / Keras
- OpenCV
